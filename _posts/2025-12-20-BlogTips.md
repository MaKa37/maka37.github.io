# Github Blog Management Tips

## Layout

### Menu Managements
- 메뉴와 포스트를 관리하는 방법
  - `_posts` 폴더에 게시글을 업로드하고 front matter 데이터로 관리하는 것이 핵심.
  - `_posts` 폴더가 없을 시 경로(`username.github.io/_posts`)에 폴더를 생성한다.

```yaml
Menu
 ├─ Posts        (Page)
 ├─ Series       (Page)
 ├─ Categories   (Page)
 └─ About        (Page)

Post
 ├─ categories
 └─ series
```

#### Post Managements
Post: 
- 게시글을 의미하며 날짜 기반 정렬
- RSS/최근 글
- 카테고리·태그·시리즈는 front matter 데이터로 관리한다.

```yaml
# File Name EX: YYYY-MM-DD-title.md
_posts/
 ├─ 2025-01-10-git-basic-1.md
 ├─ 2025-01-12-git-basic-2.md
 └─ 2025-02-01-git-basic-3.md
```

#### Front Matter Datas

```md
---
title: "Git 기초 1"
categories: [git]
series: git-basic
order: 1
---
```
